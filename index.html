y·Can·ta

/ and sing! /


A free intuitive song manager with type-to-search presentation mode and PDF export for text and chords - designed to quickly create, manage, present, and export both songs and songbooks offline or online on any browser or device.



Demo

You can try yCanta as a local copy with a prepopulated set of public domain songs below.

A song manager built as a:

  • Offline capable PWA
  • Built on PouchDb/CouchDb
  • Using pdfKit
